# Κλήση ECLiPSe από την java

Στα VM arguments θα πρέπει να προστεθεί το:

-Declipse.directory=<ECLiPSe directory>

όπου <ECLiPSe directory> είναι ο φάκελος στον οποίο έχει εγκατασταθεί το ECLiPSE
για παράδειγμα σε μια τυπική εγκατάσταση του ECLiPSe είναι (με τα εισαγωγικά): "C:\Program Files\ECLiPSe 6.1"
