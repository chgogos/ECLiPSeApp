# Κλήση ECLiPSe CLP από την java

Κατά την εκτέλεση του java προγράμματος (ECLiPSeTest.java) στα VM arguments θα πρέπει να προστεθεί το:

-Declipse.directory=ECLiPSe_directory

όπου ECLiPSe_directory είναι ο φάκελος στον οποίο έχει εγκατασταθεί το ECLiPSE CLP


για παράδειγμα σε μια τυπική εγκατάσταση στα Windows του ECLiPSe CLP το ECLiPSe_directory είναι (με τα εισαγωγικά): "C:\Program Files\ECLiPSe 6.1"
